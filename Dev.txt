--------- SETUP ---------

npm init -y

npm install autoprefixer --save-dev
npm install concat --save-dev
npm install node-sass --save-dev
npm install npn-run-all --save-dev
npm install postcss-cli --save-dev
npm install live-server --save-dev

--------- COMMANDS ---------

1) run live server and compile sass

npm run start


2) Compile sass and compile CSS for build

npm run build:css
