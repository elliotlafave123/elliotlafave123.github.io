The task:

An image of the task.

Part 1:

This code gets the number of students in the class, the highest and lowest marks and the average mark formatted to two decimal places.

  • The max function loops over the array and checks for the largest number
  • The min function does the opposite

An image of part 1.

Part 2:

This code returns a dictionary with the mark ranges, the number of marks within that range and a asterisks counter to tally them up.

  • The get asterisks function takes the number and returns the same number of asterisks.

  • The get occourances in range function checks for how many times a number appears within a range by checking if the number is between a min and max value provided to the function.

An image of part 2.

Part 3:

This code displays the calculated pass mark for the class where at least 60% of the class has passed the exam (to the nearest 10).

The students over function takes in a number on each loop and checks to see how many students have passed.

When the number of students who have passed is closest to 60 then the got_pass_mark function returns true and the last value passed into the students_over function is then the pass mark.

An image of part 3.

The result of running the program

An image of the program running in the console.

Conclusion

I found it easy to get the min, max of the arrays data and to calculate the averages.

I found the calculation of the pass mark to be the most difficult bit as it required some more complex calculations and working with multiple loops to get the desired result.

I found it quite easy to complete this activity as we did a similar task in the classroom sessions.
